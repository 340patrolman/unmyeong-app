# 운명 분석 (unmyeong-app)

사주명리 기반 자기 이해·성찰 도구. 계산은 코드가, 해석은 사람이.

## v0.2 (2026-07-08)
- 십신(十神) 산출: 천간 + 지지 본기
- 오행 왕쇠: 지장간 분일 + 월지 1.5배 가중 (삼합·회국 미반영 — 다음 판 예정)
- 대운: 방향(양남음녀 순행 규칙) + 대운수(3일=1년) + 현재 대운 표시
- 프로필에 성별 추가, v0.1 저장 프로필 자동 재계산
- 검증 세트 16건으로 확장

## v0.1 (2026-07-08)
- 사주 원국 산출 엔진: 절기 기준 년·월주 (1920~2050 절입 분 단위 테이블 내장), 60갑자 일주, 진태양시 보정 시주, 자시 일진 교체(전통)
- 가족 프로필 저장 (브라우저 localStorage — 외부 전송 없음)
- 내장 자가 검증 8케이스 (기준 원국 2건 + 절기·년경계·자시 경계)

## 개발 규율
1. 원국·천문 계산은 코드로만 (AI 생성 금지)
2. 고전 인용은 원문 대조 후 탑재
3. 매 편집 후 구동 검증 (jsdom smoke)
4. 생년월일 데이터는 기기 내 저장 원칙
5. 참고용 성찰 도구 고지 상시 표기

## 파일
- `index.html` — 앱 본체 (단일 파일)
- `saju_engine.js` — 계산 엔진 (모듈판)
- `test_saju.js` — node 검증 스크립트
- `jieqi_table.json` — 절기 전환 테이블 (sxtwl 산출)
- `expected_*.json` — 검증 세트 (ground truth)
